    linux下ELF文件GCC编译相关保护

    网络安全 PWN 网络安全
    创建时间:2019-11-23 00:37
    阅读:
    1. NX：
    2. Canary：
    3. PIE：
    4. RELRO：

    PWN基础

    NX：

    • -z execstack 关闭
    • -z noexecstack 开启
    • 不让执行栈上的数据，于是JMP ESP就不能用了

    Canary：

    • -fno-stack-protector 关闭
    • -fstack-protector 开启
    • -fstack-protector-all 全开启
    • 栈里插入cookie信息

    PIE：

    • -no-pie 关闭
    • -pie 开启
    • 地址随机化，另外打开后会有get_pc_thunk

    RELRO：

    • -z norelro 关闭
    • -z lazy 部分开启
    • -z now 完全开启
    • 对GOT表具有写权限
