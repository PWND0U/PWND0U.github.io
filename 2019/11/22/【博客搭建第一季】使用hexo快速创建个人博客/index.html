    【博客搭建第一季】使用hexo快速创建个人博客

    成长记 博客搭建 新手指引
    创建时间:2019-11-22 15:59
    阅读:
    1. 下载nodejs
    2. 开始安装hexo
    3. 开始初始的blog之路
    4. 部署到GitHub上
      1. 创建自己的仓库
    5. hexo快速使用命令
  • Quick Start
    1. Create a new post
    2. Run server
    3. Generate static files
    4. Deploy to remote sites

    • 下载nodejs

    https://nodejs.org/en/

    • 下载安装后检查自己的版本号

      node -v

    • 查看npm版本号

      npm -v

    • 以上两步全部完成后切换至阿里源

      npm install -g cnpm --registry=https://registry.npm.taobao.org

    开始安装hexo

    cnpm install -g hexo-cli

    开始初始的blog之路

    1. 新建自己的blog文件夹、
    • 我的在/usr/local/blog

    1. 进入自己的blog文件夹

      //下载初始的blog模板
sudo hexo init

    2. 下载完成,启动hexo并验证

      hexo s

    3. 新建一个文档

      hexo new "My New Post"

    4. 清理目前的文件

      hexo clean

    5. 重新生成静态资源

      hexo generate

    6. 重启服务查看各个写的文档

      hexo s

    部署到GitHub上

    创建自己的仓库

    1. 安装hexo-deployer-git插件
    cnpm install --save hexo-deployer-git

    1. 修改blog文件夹下的_config.yml文件的最后几行

      deploy:
 type: git
 repo: https://github.com/<username>/<project>
 # example, https://github.com/hexojs/hexojs.github.io
 branch: gh-pages

    2. 推送到自己的仓库

      hexo d

    hexo快速使用命令

    Quick Start

    Create a new post

    $ hexo new "My New Post"

    Run server

    $ hexo server

    Generate static files

    $ hexo generate

    Deploy to remote sites

    $ hexo deploy
